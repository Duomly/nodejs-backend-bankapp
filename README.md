## Installation

First you need to create a PostgreSQL database and connect with your credentials in DatabaseProvider.ts and Migrate.ts file.

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Migrations

```bash
$ npm run migrate up

$ npm run migrate down

```
